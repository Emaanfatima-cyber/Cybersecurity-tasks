# Cybersecurity-tasks
A secure user management system built with Node.js and Express, implementing authentication, password hashing, input validation, and security best practices including JWT, bcrypt, and Helmet for enhanced web application security.
# Cybersecurity Project

## Features:
- Signup/Login system
- JWT Authentication
- bcrypt password hashing
- Helmet security headers
- Input validation

## Installation:
npm install
npm start

## Security Improvements:
- XSS protection
- SQL injection prevention
- Password hashing
- Secure headers
